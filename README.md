# Northstar

Sistema operativo personal. Reúne clima, mercado, agenda, viajes, vehículo,
mascota y research en una sola interfaz calma.

## Estado

- Datos reales sin API key: clima (Open-Meteo), BTC (CoinGecko), dólar (dolarapi.com).
- Persistencia local (localStorage) en todos los módulos editables.

## Próximos pasos

- Google Calendar → agenda real.
- Capa de IA (briefing, research, captura rápida) sobre datos reales.

Hecho con Next.js. Deploy en Vercel.
